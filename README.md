# RH — Indicadores de Pessoas (Power BI)

## Objetivo
Construir um painel analítico para acompanhamento de indicadores de RH,
permitindo análise de headcount, turnover e movimentações de colaboradores
por área, unidade e período.

## Contexto do Projeto
Projeto de caráter demonstrativo e educacional, utilizando dados públicos
ou fictícios, com foco em boas práticas de modelagem, ETL e indicadores
gerenciais de pessoas.

## Escopo Analítico
- Cadastro de colaboradores
- Movimentações (admissões e desligamentos)
- Estrutura organizacional

## Principais Indicadores
- Headcount
- Turnover (%)
- Admissões x desligamentos
- Distribuição por área e unidade
- Tempo médio de permanência

## Ferramentas Utilizadas
- Power BI Desktop
- Power Query (M)
- Excel / CSV

## Observações
Este projeto utiliza apenas dados públicos ou fictícios
e não representa dados reais de nenhuma organização.
